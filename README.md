Habit Forge — Трекер привычек на 7 языках
Habit Forge — это коллекция из семи независимых реализаций трекера привычек, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода и предоставления удобного инструмента для формирования полезных привычек с расширенными возможностями.

✨ Уникальные возможности (общие для всех версий)
📝 Управление привычками: создание, редактирование, удаление.

📅 Периодичность: ежедневные, еженедельные, ежемесячные привычки.

✅ Отметка выполнения за сегодня (или за любую дату).

📊 Статистика и прогресс:

Общее количество выполнений.

Процент выполнения за последние 7 дней.

Текущая серия (streak) непрерывных выполнений.

💾 Сохранение данных в JSON-файл.

🔍 Просмотр списка привычек с индикацией статуса (выполнена сегодня/нет).

🖥️ Интерфейсы:

Командная строка (CLI) с интерактивным меню.

Графический интерфейс (GUI) в Python (Tkinter), Java (Swing), C# (WinForms).

Веб-интерфейс в PHP.

🌐 Кроссплатформенность — работает на Windows, macOS, Linux.

🚀 Быстрый старт
Установка зависимостей (при необходимости)
Python
bash
pip install tkinter  # (опционально для GUI)
JavaScript (Node.js)
bash
npm install commander
Go
bash
go get -u
Rust
bash
cargo add clap serde_json chrono
Java
Не требует внешних библиотек (используется стандартный java и javax.swing).

C#
Не требует внешних библиотек (используется System.Windows.Forms и System.Text.Json).

PHP
Не требует внешних библиотек.

Использование (общий синтаксис)
CLI (все версии):

bash
habit_tracker add "Утренняя зарядка" --description "10 минут" --frequency daily
habit_tracker list
habit_tracker check 1
habit_tracker stats
habit_tracker delete 1
Интерактивный режим (CLI):
Запустите без аргументов – откроется меню.
